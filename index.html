Projects

  • Phish in the Web: For the project "Phish in the Web" in the Applied Machine Learning in Python course, we developed a machine learning-based phishing website detection system. Leveraging a dataset of 10,000 URLs, we extracted key features such as URL length, subdomain level, and the presence of special characters to distinguish phishing websites from legitimate ones. We implemented several classification models, including Random Forest, SVM, MLP, and Logistic Regression, and achieved a highest AUC score of 0.982 using the Random Forest model. To enhance usability, we designed an interactive demo using Gradio, allowing users to input a URL and receive real-time predictions on its safety. This project provided hands-on experience in feature engineering, data preprocessing, model optimization, and creating user-friendly interfaces, addressing a critical challenge in cybersecurity.
  • Telco Customer Churn: In our Telco Customer Churn project, we analyzed a fictional dataset from IBM to identify patterns in customer behavior and predict churn using statistical testing and machine learning techniques. The dataset, consisting of 7,043 data points and 33 features, included demographics, service usage, and payment details. We started with data cleaning and feature selection to remove irrelevant variables, followed by statistical testing to uncover key factors contributing to churn. Insights revealed that churn was primarily driven by issues such as poor customer support and competitive offers with higher download speeds. To enhance prediction accuracy, we developed three machine learning models using Random Forest classification. Our second model, which balanced the dataset through undersampling, achieved the best overall performance with precision, recall, and F1 scores of approximately 0.76. The most influential features in predicting churn included customer tenure, total charges, and monthly charges. Based on these findings, we proposed a comprehensive solution involving preemptive targeting of high-risk customers, improved customer support, and competitive incentives to reduce churn effectively.
